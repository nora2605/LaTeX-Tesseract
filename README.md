# LaTeX-Tesseract
A tesseract in LaTeX tikz-cd

It's colored as one of the minimal sets of Hamming-Balls needed to cover all of {0, 1}^4 and also labeled as such.

(1000 = inner, 0100 = down, 0010 = right, 0001 = deep)
